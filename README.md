# polarimetric-slope-sensing
Implementation of the Polarimetric Slope Sensing technique of Zappa et al. [2008]
...with many updates and tweaks since by N. Laxague

## Overview

The original implementation is provided in _MATLAB_, though that will take some reading and interpretation to sort out how things should run.

The actively-maintained implementation is provided in _Python_. Check out the language-specific _README.md_ in that directory and examine _PSS_walkthrough.ipynb_ for a look at the full pipeline.

**This workflow is optimized for Division of Focal Plane (DoFP) polarimeters (e.g., the Sony Polarsens)**

## Our Data Example

If you want to jump straight to some example data we collected in the field (Air-Sea Interaction Tower, or ASIT; October 2019), you can find a set of raw polarized light intensity fields at DOI: [10.5281/zenodo.20361228](https://doi.org/10.5281/zenodo.20361228).
